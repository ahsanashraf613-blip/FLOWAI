Frequently Asked Questions

What counts as a "task" in FlowAI?▼
A task is a single action performed by a workflow step — like sending an email, updating a record, or making an AI call.
Can I upgrade or downgrade at any time?▼
Absolutely. You can upgrade, downgrade, or cancel at any time. Changes take effect at the start of your next billing cycle.
Is there a free trial for the Pro plan?▼
Yes! Every new account gets a 14-day free trial of the Pro plan — no credit card required.
How does FlowAI handle data security?▼
SOC 2 Type II certified, GDPR compliant. All data encrypted in transit (TLS 1.3) and at rest (AES-256).

Quick Start Guide

Welcome to FlowAI! This guide walks you through creating your first automated workflow in under 5 minutes.

Step 1: Install the SDK

# npm npm install @flowai/sdk

Step 2: Initialize the client

import { FlowAI } from '@flowai/sdk'; const flow = new FlowAI({ apiKey: 'your-api-key' });

Step 3: Create your first workflow

const wf = await flow.workflows.create({ name: 'Lead Alert', trigger: { type: 'webhook' }, actions: [ { type: 'ai.summarize' }, { type: 'slack.message', channel: '#sales' } ] });
