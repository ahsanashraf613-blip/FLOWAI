Frequently Asked Questions

What counts as a "task" in FlowAI?▼
A task is a single action performed by a workflow step — like sending an email, updating a record, or making an AI call. Each step in a workflow execution counts as one task.
Can I upgrade or downgrade at any time?▼
Absolutely. You can upgrade, downgrade, or cancel your plan at any time. Changes take effect at the start of your next billing cycle.
Is there a free trial for the Pro plan?▼
Yes! Every new account gets a 14-day free trial of the Pro plan — no credit card required. After the trial, you can choose to subscribe or revert to the free Starter plan.
How does FlowAI handle data security?▼
FlowAI is SOC 2 Type II certified and GDPR compliant. All data is encrypted in transit (TLS 1.3) and at rest (AES-256). We never sell your data to third parties.

Quick Start Guide

Welcome to FlowAI! This guide walks you through creating your first automated workflow in under 5 minutes.

Step 1: Install the FlowAI SDK

# Using npm npm install @flowai/sdk # Using yarn yarn add @flowai/sdk

Step 2: Initialize the client

import { FlowAI } from '@flowai/sdk'; const flow = new FlowAI({ apiKey: 'your-api-key', workspace: 'your-workspace-id' });

Step 3: Create your first workflow

const workflow = await flow.workflows.create({ name: 'New Lead Notification', trigger: { type: 'webhook', event: 'form.submitted' }, actions: [ { type: 'ai.summarize', input: '{{trigger.body}}' }, { type: 'slack.message', channel: '#sales' } ] });

Your workflow is now live! For more complex examples, explore the Workflow Library or the API Reference.
